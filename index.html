About "Javadoc to Markdown"

		

		

		  
Simple

		  
488 lines of well-documented code is all you need to convert your Javadoc to Markdown.

		  
Flexible

		  
You can convert not only from Javadoc but also from PHPDoc, JSDoc etc. using this library.

		  
Open source

		  
"Javadoc to Markdown" is available under the terms of the Apache License 2.0 for your use.

		

		

		  
Portable

		  
Use the converter on any device with a modern web browser or call it from your Node.js tools.

		  
Lightweight

		  
javadoc-to-markdown.js, which does the conversion, is only 5.5 KB compressed.

		  
Community

		  
We welcome all contributions, whether these are questions, issues, code fixes or improvements.
